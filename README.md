# 🎮 플레이랩 — 웹게임 놀이터

웹게임을 **즐기고 · 직접 만들고 · 만든 게임을 실행**할 수 있는 사이트입니다.
설치가 필요 없는 순수 HTML/CSS/JavaScript로 만들어졌습니다.

## 실행 방법

### 방법 1 — 가장 간단 (더블클릭)
`index.html` 파일을 크롬 또는 엣지 브라우저로 여세요.

### 방법 2 — 로컬 서버 (권장)
일부 브라우저는 `file://`에서 하위 페이지 로딩을 제한합니다. 문제가 있으면 서버로 여세요.

```bash
# 이 폴더에서 실행
python -m http.server 8000
```
그다음 브라우저에서 `http://localhost:8000` 접속.

## 기능

| 탭 | 설명 |
|----|------|
| 🕹️ 게임 즐기기 | 내장 게임 5종 (스네이크 · 2048 · 메모리 카드 · 틱택토 · 벽돌깨기) |
| 🛠️ 게임 만들기 | HTML/CSS/JS 코드 에디터 + 실시간 미리보기 |
| 📁 내 게임 | 직접 만든 게임 저장 · 실행 · 편집 · 삭제 |

- 만든 게임은 브라우저 **localStorage**에 저장됩니다 (같은 브라우저에서 계속 유지).
- 모든 게임은 **격리된 iframe(sandbox)** 에서 실행되어 안전합니다.

## 폴더 구조
```
index.html          메인 페이지
css/style.css       디자인
js/games.js         내장 게임 목록 + 저장소
js/creator.js       게임 만들기 에디터
js/app.js           탭 전환 · 카드 렌더링 · 게임 실행
games/              내장 게임 HTML 파일들
```

## 새 내장 게임 추가하기
1. `games/` 폴더에 독립 실행 가능한 `.html` 파일을 추가합니다.
2. `js/games.js`의 `BUILTIN_GAMES` 배열에 항목을 추가합니다.
